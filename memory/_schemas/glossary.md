# Jarvis 共享术语表

> 这是 Jarvis 和主人之间的领域语言。每个术语有精确的、不可被误解的定义。用一个词代替二十个词的解释。

## 记忆架构

| 术语 | 定义 |
|------|------|
| **永久层** | `memory/core/`。永不过期。身份、价值观、决策、人际关系、工具箱。 |
| **热层** | `memory/daily/`。最近 90 天日志。启动时加载最近 3 天。 |
| **温层** | `memory/archive/`。90 天前的日志。仅按需搜索。 |
| **临时层** | `memory/scans/`。系统扫描报告。30 天后过期。 |

## 搜索

| 术语 | 定义 |
|------|------|
| **RRF 融合** | Reciprocal Rank Fusion。向量搜索结果 + BM25 关键词搜索结果 → 合并排名。 |
| **时间衰减** | 搜索结果按时间加权。30天内×1，30-90天×0.8，>90天×0.5。 |
| **重要性加权** | High 结果 ×1.2，Low 结果 ×0.8。 |

## 采集

| 术语 | 定义 |
|------|------|
| **12 步入职** | 从零了解主人的标准流程：GitHub→系统扫描→浏览器历史→搜索词→Git→消费→AI对话→深度分析。 |
| **Stop Hook** | Claude Code 会话结束时自动触发，记录会话摘要 + 自动向量化。 |
