# Public parity matrix

This release is a privacy-safe capability port, not a mirror of the private repository. The private tree was used only for read-only architectural review.

| Category | Included or disposition |
| --- | --- |
| Safe now | External Vault boundary; offline bootstrap/doctor; capture → distill → proposal → approval → apply; morning skeleton; recommendation-only routing; existing SCAR/BM25/research radar |
| Genericized later | Project/thread concepts, automation scheduling, desktop release gates |
| Private-only | Memory, daily evidence, control database, ledgers, identity/relationship/decision/project data, server inventory, credentials, cookies, reports |
| Deferred due to coupling | Electron app, War Room/control DB, Codex ownership integration, workstation scans, provider-backed distillation |

The machine-readable source of truth is [`public-parity.json`](./public-parity.json).
