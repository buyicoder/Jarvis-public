# Jarvis 1.0.0-preview.1

Status: **Developer Preview**

## Compatibility

- Apple Silicon (`arm64`)
- macOS 12 or newer
- Packaged app: Node.js not required
- Source workflow: Node.js 22.13 or newer

## Distribution boundary

The preview is intended for the [`v1.0.0-preview.1` GitHub prerelease](https://github.com/buyicoder/Jarvis-public/releases/tag/v1.0.0-preview.1) with an arm64 DMG, arm64 ZIP, and aggregate `SHA256SUMS` file. Tagged source belongs at `https://github.com/buyicoder/Jarvis-public/archive/refs/tags/v1.0.0-preview.1.zip`. If the release or tag is absent, the corresponding artifact has not been published. Verify the checksum before installation. This build is ad-hoc signed, not Developer ID signed, not notarized, not a stable release, and has no auto-update.

## Included preview surface

- sandboxed local desktop and Project War Room
- external Vault with approval-gated memory writes
- canonical local control database and recovery commands
- opt-in synthetic demo that refuses non-empty user state
- local retrieval plus disabled-by-default provider, activity, Codex, and network adapters

No private memory, runtime instance, account, credential, browser row, server record, real thread record, or personal report is included.

## Install and first run

Follow the [Developer Preview guide](../developer-preview.md) to download, verify, install, use Finder's per-app Gatekeeper **Open** flow, initialize an optional synthetic demo, locate data, uninstall, troubleshoot, and roll back safely.

## Known limitations

- Gatekeeper will warn because the preview is not Developer ID signed or notarized.
- Updates and downgrades are manual.
- Preview data formats and UI may change before a stable release.
- Optional integrations remain unavailable until explicitly configured.
